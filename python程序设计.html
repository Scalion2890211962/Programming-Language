Python知识点：变量与打印

1. 变量：Python中的变量不需要声明类型，直接赋值即可，比如 name = "小明"。

2. 打印函数print()：用于输出内容到控制台，是Python最基础的输入输出函数。

3. 字符串拼接：使用 + 号可以将多个字符串连接在一起，比如 "你好，" + name + "！"。

4. 命名规则：变量名只能包含字母、数字、下划线，且不能以数字开头。
